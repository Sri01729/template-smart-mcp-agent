Template Smart MCP Agent

A Mastra template for smart AI agents that dynamically discover, install, and use Model Context Protocol (MCP) servers at runtime.

ğŸš€ Features

  • Dynamic MCP Server Management: AI agent can discover and add MCP servers from Smithery registry
  • Runtime Configuration: Servers are added directly to the codebase via file modification
  • Persistent Storage: Server configurations persist across restarts
  • Smithery Integration: Seamless integration with Smithery MCP server registry
  • File System Access: Built-in filesystem MCP server for project file operations

  • Node.js 20.9.0 or higher
  • npm or pnpm package manager
  • API keys for AI providers (OpenAI, Anthropic, etc.)

ğŸ¤– Agent Capabilities

The App Store Agent provides the following tools:

Core Tools

  • discover-servers: Search for MCP servers on Smithery registry
  • add-mcp-server: Dynamically add a server to the MCP configuration
  • list-servers: List all currently configured MCP servers

MCP Server Tools

The agent has access to tools from configured MCP servers:

  • Filesystem tools: File and directory operations
  • GeekNews tools: Technology news and articles
  • Cultural Heritage tools: Cultural and historical information
  • Sequential Thinking tools: Advanced reasoning capabilities
  • A-Share Stock Market tools: Chinese stock market data
  • Visual Reasoning tools: Image analysis and reasoning

MCP Server Management

// Add a new MCP server
await addMcpServerTool.execute({
  context: {
    name: "websearch",
    serverId: "@Aas-ee/open-websearch",
    description: "Web search capabilities"
  }
});

// List configured servers
await listServersTool.execute({ context: {} });

Tool Schemas

All tools use Zod schemas for input validation:

// Add MCP Server Tool
{
  name: z.string().min(1),
  serverId: z.string().min(1),
  description: z.string().optional()
}

// Discover Servers Tool
{
  query: z.string().optional(),
  pageSize: z.number().optional()
}

ğŸ”„ Workflow

  1. Discovery: Use discover-servers to find MCP servers on Smithery
  2. Installation: Use add-mcp-server to add servers to the configuration

  textEditor: {
    command: 'npx',
    args: ['-y', '@modelcontextprotocol/server-filesystem', path.join(process.cwd(), '..', '..')],
  },
  // Agent-added servers appear here
};

ğŸ› Troubleshooting

Common Issues

MCP Server Connection Failures

  • Check if the server ID exists on Smithery
  • Verify network connectivity
  • Check server-specific requirements

File System Access Issues

  • Ensure proper file permissions
  • Check path resolution in bundled environment

API Key Issues

  • Verify API keys are set in .env
  • Check API key permissions and quotas

Debug Mode

Enable debug logging by setting:

TASKMASTER_LOG_LEVEL=debug
